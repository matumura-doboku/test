1. 目的

本システムは、マクロな視点からデータを可視化し、都市の傾向や現状を把握することを目的としています。

2. 主な機能

  • 可視化機能: 地図上に人口、交通量、施設情報など様々なデータを重ねて表示し、視覚的に分析できます。
  • グラフ作成機能: 可視化機能で分析・抽出したデータをもとに、報告書用のグラフを作成できます。
  • データ加工機能: 国土交通省データプラットフォーム等のAPIを通じてデータを取得し、本システムで利用可能な形式に加工・統合します。

3. 可視化機能の使用方法

  1. ホームタブ: 「住所検索」機能を使用して、分析したい地域へ地図を移動します。
  2. 可視化タブ: 「表示項目」を選択してデータを地図に表示させます。必要に応じてフィルタ条件（上位%や数値範囲）を設定し、対象を絞り込みます。
  3. レポートタブ: 指定した範囲内のデータについて、上位セルの抽出、最大・最小・平均値の確認を行います。結果はCSVファイルとして出力可能です。
  4. 概要タブ: 現在の表示状態を「概要」として保存し、そのままPowerPoint (PPTX) 形式で画面イメージを出力できます。

4. グラフ作成機能の使用方法

  1. CSVアップロード: 可視化機能の「レポートタブ > 条件検索」から出力したCSVファイルを読み込みます。
  2. 項目選択: グラフ化したいデータ項目を選択します。
  3. 描画: 「グラフを描画」ボタンを押してグラフを表示させます。
  4. 加工: 必要に応じて注釈や強調などの加工を行います。
  5. 出力: 作成したグラフをPowerPoint (PPTX) 形式で出力します。

5. データ加工機能の使用方法

  1. API選択: 使用したいAPIを選択します。
  2. 都道府県単位で取得する地域を選択します
  3. 「エクスポート」ボタンからデータを可視化システムに贈ります
